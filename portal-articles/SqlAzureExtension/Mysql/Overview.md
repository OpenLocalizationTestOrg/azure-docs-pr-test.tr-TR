# <a name="azure-database-for-mysql-documentation"></a>MySQL Belgeleri için Azure Veritabanı

MySQL için Azure Veritabanı, açık kaynak MySQL Server altyapısını temel alan ilişkisel bir veritabanı hizmetidir.  Bu; kritik iş yüklerini teslim edebilen, tahmin edilebilir performans ve dinamik ölçeklenebilirlik sunan, tam olarak yönetilen bir hizmet olarak veritabanıdır. MySQL için Azure Veritabanı ile açık kaynak araçları ve seçtiğiniz platformu kullanarak uygulamalar geliştirin. Hızlı başlangıç kılavuzlarımız, öğreticilerimiz ve örneklerimizle, MySQL için Azure Veritabanı’nı kullanmayı öğrenin.
