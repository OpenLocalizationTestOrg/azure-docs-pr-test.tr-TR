# <a name="virtual-machines-documentation"></a>Sanal makineler belgeleri

- [Windows sanal makinesi oluşturma](/azure/virtual-machines/windows/quick-create-portal/?WT.mc_id=UIHelpPilot)
- [Linux sanal makinesi oluşturma](/azure/virtual-machines/linux/quick-create-portal/?WT.mc_id=UIHelpPilot)
