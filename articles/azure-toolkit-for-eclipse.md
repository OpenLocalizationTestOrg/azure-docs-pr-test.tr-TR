# <a name="azure-toolkit-for-eclipse"></a>Eclipse için Azure Araç Seti
Eclipse için Azure araç seti, şablonları ve kolayca oluşturun, geliştirme, test ve Eclipse tümleşik geliştirme ortamı (IDE) kullanarak Azure uygulamalarını dağıtmak için kullanabileceğiniz işlevsellik sağlar. Kaynak kodu altında kullanılabilir bir açık kaynak projesi olan [MIT lisansı projenin sitesinden](https://github.com/microsoft/azure-tools-for-java).

> [!NOTE]
> Intellij IDE için bir Azure Araç Seti yoktur. Daha fazla bilgi için bkz: [Intellij için Azure Araç Seti](azure-toolkit-for-intellij.md).
