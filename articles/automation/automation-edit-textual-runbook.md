### <a name="to-retrieve-the-contents-of-a-runbook-using-windows-powershell"></a><span data-ttu-id="339a9-184">Windows PowerShell kullanarak bir Runbook'un içeriğini almak için</span><span class="sxs-lookup"><span data-stu-id="339a9-184">To Retrieve the Contents of a Runbook Using Windows PowerShell</span></span>
<span data-ttu-id="339a9-185">Aşağıdaki örnek komutlar bir runbook için komut almak ve bir komut dosyasına kaydetmek nasıl gösterir.</span><span class="sxs-lookup"><span data-stu-id="339a9-185">The following sample commands show how to retrieve the script for a runbook and save it to a script file.</span></span> <span data-ttu-id="339a9-186">Bu örnekte Taslak sürümü alınmaktadır.</span><span class="sxs-lookup"><span data-stu-id="339a9-186">In this example, the Draft version is retrieved.</span></span> <span data-ttu-id="339a9-187">Bu sürüm değiştirilemez rağmen runbook'un yayımlanan sürümünü almak mümkündür.</span><span class="sxs-lookup"><span data-stu-id="339a9-187">It is also possible to retrieve the Published version of the runbook although this version cannot be changed.</span></span>

    $runbookName = "Sample-TestRunbook"
    $scriptPath = "c:\runbooks\Sample-TestRunbook.ps1"

    $runbookContent = $runbookDefinition.Content

    Out-File -InputObject $runbookContent -FilePath $scriptPath
