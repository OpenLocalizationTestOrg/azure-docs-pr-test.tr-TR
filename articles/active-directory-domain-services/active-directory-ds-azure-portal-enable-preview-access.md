# <a name="enable-the-preview-experience-to-administer-your-managed-domain-from-the-azure-portal"></a>Azure Portalı'ndan yönetilen etki alanınızı yönetmek Önizleme deneyimi etkinleştir

## <a name="your-existing-managed-domain-is-not-shown-in-the-azure-portal-preview-experience"></a>Mevcut yönetilen etki alanınızı Azure portal (Önizleme) deneyimi gösterilmez
Var olan bir sahip [Azure AD etki alanı Hizmetleri](active-directory-ds-overview.md) Klasik Azure portalı kullanılarak oluşturulmuş yönetilen etki alanı. Azure Portalı'na gidin, bu varolan yönetilen etki alanı (Önizleme) Azure AD etki alanı Hizmetleri dikey penceresini listelenmez.

![Mevcut yönetilen etki alanı Azure Portalı'nda gösterilmez](./media/getting-started/missing-domain-services-instance.png)

Mevcut yönetilen etki alanınızı üzerinden yeni Azure portalına geçirilmemiş olmasıdır. Mevcut yönetilen etki alanınız için yeni Azure portalına (Önizleme) deneyimi sağlamak için bize başvurabilirsiniz.


## <a name="join-the-preview-for-the-azure-portal-experience-for-azure-ad-domain-services"></a>Azure AD etki alanı Hizmetleri için Azure portal deneyimi için Önizleme katılma
Yönergeleri izleyerek [Bize Ulaşın](active-directory-ds-contact-us.md) sayfasında ve e-posta gönderin. Mevcut yönetilen etki alanınız için Azure portal Önizleme etkinleştirmek mutlu duyuyoruz.

> [!NOTE]
> Yeni Azure portal deneyimi Azure AD etki alanı Hizmetleri için genel olarak kullanılabilir (GA) olduğunda, tüm mevcut yönetilen etki alanları otomatik olarak geçirilir. Bu noktada, Azure portalından mevcut yönetilen etki alanınızı yönetmek mümkün olacaktır.
