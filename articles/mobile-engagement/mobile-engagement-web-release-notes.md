# <a name="azure-mobile-engagement-web-sdk-release-notes"></a>Azure Mobile Engagement Web SDK sürüm notları
## <a name="202-10182016"></a>2.0.2 (10/18/2016)
* Özel (Safari) gözatma sabit kilitlenme.
* Sabit kilitlenme tarayıcılarında tanımlama bilgileri devre dışı ile.

## <a name="201-6102016"></a>2.0.1 (6/10/2016)
* Internet Explorer 8 ve Internet Explorer 9 SDK'yı devre dışı.
* Sabit Opera web tarayıcısı algılama.

## <a name="200-672016"></a>2.0.0 (6/7/2016)
* Azure Mobile Engagement yayımladı.
* Güncelleştirilmiş böylece `appId` yapılandırma bağlantı dizesini yapılandırma tarafından değiştirilir.
* Etkinleştirmek ve aracıyı devre dışı bırakmak için API'ler eklendi.
* Güvenlik geliştirmeleri yapıldı.
* Yerel JSON API'leri tarayıcılardan kullanacak şekilde güncelleştirildi.
* Cihazlar arasında ileti alıp gönderen bir API kaldırıldı.
* Kullanım dışı XMPP bağlantıyla ilgili kaldırılan geri aramalar.
* Reach özelliği kaldırılan desteği.
