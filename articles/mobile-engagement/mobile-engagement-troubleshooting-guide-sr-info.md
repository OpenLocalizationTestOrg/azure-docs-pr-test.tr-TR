# <a name="service-request-sr-information"></a>Hizmet isteği (SR) bilgileri

## <a name="ids-applicable-identifiers-related-tooyour-issue"></a>Kimlikleri: Geçerli tanımlayıcıları tooyour sorunu ilgili:
* Uygulama Kimliği
* Kampanya kimliği
* Cihaz Kimliği
* Kullanıcı Kimliği
* Kullanıcı adı
* Uygulama bilgileri etiketi

## <a name="errors-applicable-error-information"></a>Hata: Geçerli hata bilgileri
* Merhaba sorunun nerede oluştuğunu hello API veya UI bölümün adı
* Aldığınız herhangi bir hata iletisi metni
* Hello gerçekleştirilen herhangi bir test sonuçlarını [sorun giderme kılavuzları](http://go.microsoft.com/fwlink/?LinkId=524382)

* Üretim uygulamanız konumunu indirin (veya geliştirme uygulamanızın APK/TGZ dosyaları hello)
* "AndroidManifest.xml" ve/veya herhangi bir kod parçacığını uygulamanızdan Mobile Engagement tooAzure (İleri düzey sorun giderme için) ilgili
