# <a name="windows-phone-silverlight-sdk-content"></a>Windows Phone Silverlight SDK içeriği
Bu belge listeler ve uygulamanızda SDK'sı tarafından dağıtılan içeriği açıklar.

## <a name="the-resources-folder"></a>`/Resources` Klasörü
Bu klasör, Mobile Engagement gereken tüm kaynakları içerir. Ayrıca bunları uygulamanızı uyacak şekilde özelleştirebilirsiniz.

* `EngagementIconNotification.png`: Bir bildirim sol tarafında görüntülenen marka simge bunu, marka simgesiyle değiştirin.
* `EngagementIconOk.png`: Reach "Tamam" simgesini ApplicationBar sayfaları.
* `EngagementIconCancel.png`: Reach "İptal" simgesini ApplicationBar sayfaları.
* `EngagementIconCloseLight.png`: Mobile Engagement reach bildirim Windows Phone için "Kapat" simgesini açık tema.
* `EngagementIconCloseDark.png`: Windows Phone koyu tema için bildirim Mobile Engagement'ın "Kapat" simgesini ulaşabilirsiniz.
* `EngagementConfiguration.xml`: Mobile Engagement'ın yapılandırma dosyası, burada, katılım ayarları özelleştirebilirsiniz budur (katılım bağlantı dizesi, rapor çöküyor vb.).
