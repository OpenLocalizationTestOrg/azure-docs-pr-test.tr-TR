# <a name="pricing"></a>Fiyatlandırma

## <a name="compute-cost"></a>Maliyet işlem
Azure bulut Kabuk ücretsiz sağlanan bir makinede çalışır, ancak kullanılacak ekli Azure dosya paylaşımının gerektirir. 

## <a name="storage-cost"></a>Depolama maliyeti
Bulut Kabuk varsayılan 5 GB görüntü $HOME dizininize kalıcı hale getirmek için dosya paylaşımı oluşturur. Azure dosya paylaşımları normal maliyetler doğurur.

Denetleme [Azure dosyaları maliyetleri hakkında ayrıntılı bilgi için burada](https://azure.microsoft.com/pricing/details/storage/files/).
