> [!IMPORTANT] 
> Azure oluşturmak ve kaynaklarla çalışmak için iki farklı dağıtım modeli vardır: [Resource Manager ve klasik](../../../resource-manager-deployment-model.md). Bu makalede, Klasik dağıtım modeli kullanarak yer almaktadır. Microsoft, yeni dağıtımların çoğunun Resource Manager modelini kullanmasını önerir. Resource Manager modelini kullanarak VM uzantıları hakkında daha fazla bilgi için bkz: [burada](../extensions-features.md?toc=%2fazure%2fvirtual-machines%2fwindows%2ftoc.json).
