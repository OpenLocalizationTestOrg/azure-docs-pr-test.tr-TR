# <a name="deploy-popular-application-frameworks-on-windows-using-azure-resource-manager-templates"></a>Windows Azure Resource Manager şablonları kullanarak popüler uygulama çerçevelerinin dağıtma 

İş yükleri genellikle göre tasarım çalışması için birçok kaynakları gerektirir. Azure Resource Manager şablonları daha değil yalnızca size uygulamaları nasıl yapılandırılacağını, aynı zamanda kaynakların yapılandırılmış destek uygulamaları için nasıl dağıtıldığını tanımlamak için bile mümkün kılar. Bu makalede, galerideki en popüler şablonları tanıtır ve bunları dağıtmak için Azure portal, Azure CLI veya PowerShell kullanma hakkında bilgi sağlar.
