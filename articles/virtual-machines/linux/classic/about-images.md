## <a name="working-with-images"></a>İmajlarla çalışma
Mac, Linux ve Windows için Azure komut satırı arabirimi (CLI), kullanılabilir görüntüleri Azure aboneliğinizi yönetmek için kullanabilirsiniz. Azure portalındaki bazı görüntü görevler için de kullanabilirsiniz, ancak komut satırında daha fazla seçenek verir.

Araçları kullanarak örnekler için bkz: [Linux ve Mac üzerinde ortak Azure CLI komutları](../cli-manage.md?toc=%2fazure%2fvirtual-machines%2flinux%2ftoc.json).
