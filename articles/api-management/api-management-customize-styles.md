<span data-ttu-id="086ad-109">Bir sayfadaki renk, yazı tipi, boyut, boşluklar ve stille ilgili diğer öğeler stil kuralları tarafından tanımlanır.</span><span class="sxs-lookup"><span data-stu-id="086ad-109">The colors, fonts, sizes, spacings, and other style-related elements of any page on the portal are defined by style rules.</span></span>

<span data-ttu-id="086ad-113">Ardından sağ üst köşedeki **Geliştirici portalı**'na tıklayın.</span><span class="sxs-lookup"><span data-stu-id="086ad-113">Then click on **Developer portal** on the top-right.</span></span>

<span data-ttu-id="086ad-115">Özelleştirme araç çubuğunu açmak için farenizi özelleştirme simgesinin üzerine götürün (veya simgeyi seçin) ve araç çubuğundan "stiller"e tıklayın.</span><span class="sxs-lookup"><span data-stu-id="086ad-115">To open the customization toolbar move your mouse over the customization icon (or select it) and then click "styles" from the toolbar.</span></span>

<span data-ttu-id="086ad-117">Stil kurallarını iki şekilde düzenleyebilirsiniz: varsayılan olarak gösterilen ve her yerde kullanılan tüm stil kurallarının listesine bakabilir veya **Sayfadaki bir öğeyi seçin** öğesini seçip sayfadaki herhangi bir yere tıklayarak yalnızca ilgili öğenin stillerini görebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="086ad-117">There are two main way of editing styling rules - you can look through the list of all the style rules used anywhere which is displayed by default and modify a style as needed, or you can choose **Select an element on the page** and then click anywhere on the page to see only the styles for that element.</span></span>

![Özelleştirme araç çubuğu][api-management-customization-toolbar]

<span data-ttu-id="086ad-119">Bu örnek için **Sayfadaki bir öğeyi seçin**'e tıklayın.</span><span class="sxs-lookup"><span data-stu-id="086ad-119">Click the **Select an element on the page** option for this example.</span></span>  <span data-ttu-id="086ad-120">Fareyle üzerine geldiğiniz öğeler vurgulanır ve tıkladığınızda hangi öğe stillerini düzenlemeye başlayacağınızı gösterir.</span><span class="sxs-lookup"><span data-stu-id="086ad-120">Elements now become highlighted as you hover over them with the mouse to signify what element's styles you would start editing if you clicked.</span></span> <span data-ttu-id="086ad-121">Fareyi üst bilgideki metnin üzerine götürün (genelde burada şirket adı yazar) ve tıklayın.</span><span class="sxs-lookup"><span data-stu-id="086ad-121">Move the mouse over the text in the header (typically you have the company name here), and then click it.</span></span> <span data-ttu-id="086ad-122">Stil düzenleyicisinde, adlandırılmış ve kategorilere ayrılmış bir grup stil kuralı görüntülenir.</span><span class="sxs-lookup"><span data-stu-id="086ad-122">A set of named and categorized styling rules appears within the styling editor.</span></span> <span data-ttu-id="086ad-123">Her kural seçilen öğenin bir stil özelliğini temsil eder.</span><span class="sxs-lookup"><span data-stu-id="086ad-123">Each rule represents a styling property of the selected element.</span></span> <span data-ttu-id="086ad-124">Örneğin, yukarıda seçilen üst bilgi metni için metin boyutu @font-size-h1 biçimindeyken, alternatiflerle birlikte yazı tipinin adı @headings-font-family biçimindedir.</span><span class="sxs-lookup"><span data-stu-id="086ad-124">For example, for the header text selected above, the size of the text is in @font-size-h1 while the name of the font with alternatives is in @headings-font-family.</span></span>

> <span data-ttu-id="086ad-125">[Önyükleme][bootstrap] yapmayı biliyorsanız, bu kurallar aslında geliştirici portalı tarafından kullanılan önyükleme temasının [LESS değişkenleridir][LESS variables].</span><span class="sxs-lookup"><span data-stu-id="086ad-125">If you're familiar with [bootstrap][bootstrap], these rules are in fact [LESS variables][LESS variables] within the bootstrap theme used by the developer portal.</span></span>

<span data-ttu-id="086ad-126">Şimdi üst bilgi metninin rengini değiştirelim.</span><span class="sxs-lookup"><span data-stu-id="086ad-126">Let's change the color of the heading text.</span></span> <span data-ttu-id="086ad-127">**@headings-color** alanındaki girişi seçin ve **#000000** yazın.</span><span class="sxs-lookup"><span data-stu-id="086ad-127">Select the entry in the **@headings-color** field and type **#000000**.</span></span> <span data-ttu-id="086ad-128">Bu, siyah renk için onaltılı koddur.</span><span class="sxs-lookup"><span data-stu-id="086ad-128">This is the hex code for the color black.</span></span> <span data-ttu-id="086ad-129">Bunu yaparken, metin kutusunun sonunda kare şeklinde bir renk göstergesi görürsünüz.</span><span class="sxs-lookup"><span data-stu-id="086ad-129">As you do this, you see that a square color indicator appears at the end of the text box.</span></span> <span data-ttu-id="086ad-130">Bu göstergeye tıklarsanız, bir renk seçici renk seçmenizi sağlar.</span><span class="sxs-lookup"><span data-stu-id="086ad-130">If you click this indicator, a color picker lets you to choose a color.</span></span>

<span data-ttu-id="086ad-132">Değişiklikler yapıldıkça gerçek zamanlı olarak önizlemesi gösterilir, ancak yalnızca yöneticilere görünür.</span><span class="sxs-lookup"><span data-stu-id="086ad-132">Changes are previewed in real time as you make them, but are visible only to administrators.</span></span> <span data-ttu-id="086ad-133">Bu değişikliği herkese görünür yapmak için stil düzenleyicisinde **Yayımla** düğmesine tıklayın ve değişiklikleri onaylayın.</span><span class="sxs-lookup"><span data-stu-id="086ad-133">To make these changes visible to everyone, click the **Publish** button in the styling editor and confirm the changes.</span></span>
