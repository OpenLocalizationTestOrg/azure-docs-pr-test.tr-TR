## <a name="making-a-group-available-for-end-user-self-service"></a><span data-ttu-id="60968-125">Bir grubu, son kullanıcı self servisi için kullanıma sunma</span><span class="sxs-lookup"><span data-stu-id="60968-125">Making a group available for end user self-service</span></span>
