Bu belge veri bilimi işleminde genellikle kullanılan dört farklı depolama ortamları keşfedin alınmaktadır:

* **Azure blob kapsayıcısı** veri incelediniz kullanarak [Pandas](http://pandas.pydata.org/) Python paket.
* **SQL Server** SQL kullanarak ve Python gibi bir programlama dili kullanarak veri incelediniz.
* **Tablo hive** veri incelediniz Hive sorgularını kullanma.
* **Azure Machine Learning (AML) Studio** veri incelediniz AML modüllerini kullanma.
