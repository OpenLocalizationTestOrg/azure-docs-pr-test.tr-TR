# <a name="datamarket-is-now-deprecated"></a>DataMarket artık kullanım dışı

[Microsoft DataMarket](https://datamarket.azure.com/datasets) kullanım dışı artık ve var. barındırılan Machine Learning API'ları da artık kullanım dışı şekilde.

Artık bu özellikleri (ve daha fazlasını) bulabilirsiniz [Bilişsel hizmetler API'leri](https://www.microsoft.com/cognitive-services).

## <a name="text-analytics-api"></a>Metin analizi API

Önceki DataMarket metin Analytics API müşterilerin kullanarak başlamalıdır [Bilişsel Hizmetleri metin Analytics API](https://www.microsoft.com/cognitive-services/text-analytics-api).
Bilişsel hizmetler metin Analytics API dil algılama ve konu algılama gibi ek senaryolarda yanı sıra önceki DataMarket sürüm yaptığınız tüm senaryolarını destekler.


## <a name="recommendations-api"></a>Öneriler API’si 

Önceki DataMarket önerileri API müşterilerin kullanarak başlamalıdır [Bilişsel Hizmetleri önerileri API](https://www.microsoft.com/cognitive-services/recommendations-api).
Bilişsel hizmetler önerileri API tüm önceki DataMarket sürüm vermedi senaryoları yanı sıra batch desteği ve modelleri kalitesini değerlendirmek için daha iyi çevrimdışı ölçümleri gibi ek senaryoları destekler. 

Bilişsel hizmetler önerileri API kullanmaya başlama hakkında daha fazla bilgi için bkz: [Azure Bilişsel Hizmetleri önerileri API geçiş](http://aka.ms/recomigrate).
