# <a name="azure-remoteapp-retirement"></a>Azure RemoteApp devre dışı bırakma
Biz, Azure RemoteApp bırakma onaylamak için bağlantı kurabiliyor edilmemiş çünkü bu makalede yönlendirilmiş. 

Biz ve diğer Azure aboneliği yöneticileri Azure RemoteApp 31 Ağustos 2017 üzerinde kaldırılan uyumlu olduğunu doğrulamak için bağlantı kurmaya biz [Azure RemoteApp devre dışı bırakma 12 Ağustos 2016'daduyurdu](http://aka.ms/araretirement).   Biz urgently yanıtlamak için istek [ arainfo@microsoft.com ](mailto:arainfo@microsoft.com) belirten:
1.  31 Ağustos 2017 üzerinde devre dışı bırakma ve kuruluşunuzun, Onayla
2.  Geçiş planlarınızı hakkında bize – gerekirse yardımcı olabiliriz. Geçiş seçenekleri ayrıntılı olarak [Azure RemoteApp geçiş seçenekleri](http://aka.ms/aramigration)Citrix XenApp Essentials, MyCloudIT, çerçeve, vb. gibi. 

Azure Yönetim Portalı REMOTEAPP düğümde engellenmiş, çünkü bağlantı kuramadı ve, katılım bizimle zorla gerekiyordu bu yaptığımız.  Azure aboneliği ile ilişkili e-posta adresi değil izlenmiyor veya geçerli değil çünkü bizim e-postaları almamış olabilir.  E-postaları doğrulamak için kesinlikle öneririz, Azure aboneliğinizle ilişkili adresi geçerli ve izlenen, Azure portalında gezinme ve yöneticiler abonelik doğrulama bunu yapabilirsiniz.  

Hala sorularınız varsa veya geçiş desteği?  Adresinden bize e-posta [ arainfo@microsoft.com ](mailto:arainfo@microsoft.com) veya [toplantı kitap](http://aka.ms/ericorman) bizim geçiş ile ekip şimdi.
