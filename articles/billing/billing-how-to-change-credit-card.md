# <a name="change-the-credit-card-used-to-pay-for-an-azure-subscription"></a>Azure aboneliği için ödeme yapmak üzere kullanılan kredi kartını değiştirin
Ekleme, değiştirme veya kaldırma için oturum açtığında, Azure hesabınızla ilişkili bir kredi kartı [Azure hesap Merkezi](https://account.windowsazure.com/Subscriptions) olarak [Hesap Yöneticisi](billing-subscription-transfer.md#whoisaa). 
 
<a id="addcard"></a>
## <a name="add-a-credit-card-as-a-payment-method"></a>Bir ödeme yöntemi bir kredi kartı ekleyin

1. Oturum [Azure hesap Merkezi](https://account.windowsazure.com/Subscriptions) hesabı yönetici olarak.
2. Aboneliği seçin.
3. Sayfanın sağ tarafında **Ödeme yöntemlerini yönet**’i seçin.
4. Seçin "+" Kredi kartı eklemek için.
5. Bir kredi kartı türü seçin ve gerekli bilgileri girin.
6. Seçin **sonraki**. 

Kart geçerliyse, kullanılabilir ödeme yöntemleri listesine eklenir.

Kredi kartı ekledikten sonra bir hata alırsanız, bkz: [kredi kartı reddedildi Azure kaydolma](billing-credit-card-fails-during-azure-sign-up.md).

## <a name="edit-payment-information-for-an-existing-credit-card"></a>Mevcut bir kredi kartı için ödeme bilgilerini düzenleyin
  Kredi kartınız yenilendi ve numarası aynı kalır, sona erme tarihi gibi mevcut kredi kartı bilgileri güncelleştirin. Kart kaybolduğu için kredi kartı numarası değişikliklerinizi çalınırsa ya da süresi doldu, adımları izlerseniz, [bir ödeme yöntemi bir kredi kartı eklemek](#addcard) bölümü. 

1. Oturum [Azure hesap Merkezi](https://account.windowsazure.com/Subscriptions) hesabı yönetici olarak.
2. Kredi kartına bağlı aboneliği seçin.</br> ![Seçili bir örnek abonelik gösteren ekran görüntüsü.](./media/billing-how-to-change-credit-card/selectsub.png)
3. Seçin **ödeme yöntemleri yönetmek**.</br> ![Seçili Yönet ödeme yöntemleri seçeneği gösteren ekran görüntüsü.](./media/billing-how-to-change-credit-card/changesub_new.png)
4. Ödeme yöntemi seçin.
5. Seçin **Düzenle** yanındaki kredi kartı numarası.</br> ![Ödeme yöntemini yanındaki Düzenle seçeneğini gösteren ekran görüntüsü.](./media/billing-how-to-change-credit-card/editcard_new.png)
6. Kredi kartı türü ve kart numarasının doğru olduğundan emin olun.
7. Seçin **sonraki**.

## <a name="change-the-credit-card-that-pays-your-azure-bill"></a>Değişiklik Azure faturasını ödeyen kredi kartı

1. Oturum [Azure hesap Merkezi](https://account.windowsazure.com/Subscriptions) hesabı yönetici olarak.
2. Aboneliği seçin.
3. Sayfanın sağ tarafında **Ödeme yöntemlerini yönet**’i seçin.
4. Tıklatın **kullanmak yerine** ve kullanmak istediğiniz kredi kartı seçin.

## <a name="remove-a-credit-card-from-the-account"></a>Bir kredi kartı hesabından Kaldır
1. Oturum [Azure hesap Merkezi](https://account.windowsazure.com/Subscriptions) hesabı yönetici olarak.
2. Aboneliği seçin.
3. Sayfanın sağ tarafında **Ödeme yöntemlerini yönet**’i seçin.
4. Tıklatın **silmek** silmek istediğiniz kredi kartı için.

Kredi kartınız diğer etkin Microsoft aboneliği ile ilişkili ise, Azure hesabınızı kaldıramazsınız. Microsoft ile olan tüm etkin abonelikleri kredi kartı kaldırın.
