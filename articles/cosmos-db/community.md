# <a name="community-portal"></a>Topluluk portalı
## <a name="community-spotlight"></a>Topluluk spotlight
Bize projenizi Yükselt! Bize üzerinde Azure Cosmos DB ile DocumentDB yığın gelecek nesil çalıştığınız harika proje gösterebilir ve biz, genius dünyayla paylaşmak yardımcı olur. Projenizi göndermek için e-posta adresinden bize gönderin: [ askcosmosdb@microsoft.com ](mailto:askcosmosdb@microsoft.com).

### <a name="documentdb-lumenize"></a>documentdb-lumenize
*tarafından Larry Maccherone*

Toplamalar (Group by Özet Tablo ve N boyutlu küp) ve zaman serisi dönüşümler API Azure Cosmos DB documentdb'de saklı yordamlar.

Out üzerinde kontrol [GitHub](https://github.com/lmaccherone/documentdb-lumenize) ve [npm](https://www.npmjs.com/package/lumenize).

### <a name="documentdb-studio"></a>DocumentDB Studio
*Msiexec Liu tarafından*

Bir istemci yönetimi Görüntüleyicisi/explorer Azure Cosmos DB DocumentDB API hizmeti.

Out üzerinde kontrol [GitHub](https://github.com/mingaliu/DocumentDBStudio).

### <a name="doqmentdb"></a>DoQmentDB
*Ariel Mashraki tarafından*

DoQmentDB Azure Cosmos DB üstünde MongoDB benzeri katmanı sağlayan bir Node.js promise tabanlı istemcidir.

Out üzerinde kontrol [GitHub](https://github.com/a8m/doqmentdb) ve [npm](https://www.npmjs.com/package/doqmentdb).

### <a name="typescript-api"></a>TypeScript API
*Jelmer Cormont tarafından*

TypeScript içinde yazılmış Node.js istemci çevresinde bir sarmalayıcı (düz JavaScript'te çok çalışır). Destekler `async/await` ve Basitleştirilmiş bir API.

Out üzerinde kontrol [GitHub](https://github.com/jcormont/documentdb-typescript) ve [npm](https://www.npmjs.com/package/documentdb-typescript).

### <a name="swagger-rest-api-for-documentdb"></a>DocumentDB için swagger REST API
*tarafından Howard Edidin*

Azure Cosmos DB REST API'si Swagger dosyası bir API uygulaması kolayca dağıtılabilir DocumentDB API'si.

Out üzerinde kontrol [GitHub](https://github.com/HEDIDIN/DocumentDB-REST/tree/master/DocumentDBRestApi).

### <a name="fluent-plugin-documentdb"></a>Eklenti documentdb Fluent
*Yoichi Kawasaki tarafından*

Eklenti documentdb Fluent Azure Cosmos DB DocumentDB API'sine çıktısı için bir Fluentd eklentidir.

Out üzerinde kontrol [GitHub](https://github.com/yokawasa/fluent-plugin-documentdb) ve [rubygems](https://rubygems.org/gems/fluent-plugin-documentdb).

*Daha açık kaynak üzerinde Azure Cosmos DB projeleri Bul [GitHub](https://github.com/search?p=4&q=documentdb&type=Repositories).*
