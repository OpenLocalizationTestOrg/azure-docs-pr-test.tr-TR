# <a name="web-app-on-linux-documentation"></a>Web uygulamasında Linux belgeleri
Web uygulaması Linux üzerinde şu anda genel önizlemede ve müşteriler toorun bir Linux platformunda yerel web uygulamalarını sağlar. Bu, belirli türdeki uygulamalar için daha iyi uygulama uyumluluğu için verir ve bir Linux platformuna başka bir yerde Azure App Services üzerinde barındırılan daha kolay toomigrate varolan web uygulamaları kolaylaştırır.

Müşteriler oluşturabilirsiniz web uygulamaları Linux'ta içinde yalnızca bir Linux App Service planı etkin ve bu uygulama hizmeti planı yalnızca Linux tabanlı web uygulamaları barındırabilir.
