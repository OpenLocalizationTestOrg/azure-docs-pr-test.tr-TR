# <a name="azure-project-properties"></a>Azure proje özellikleri
Azure projeniz için çeşitli ayarları hello Eclipse için Azure Araç Seti içinde ayarlanabilir.

## <a name="configuring-azure-project-properties"></a>Azure proje özellikleri yapılandırma
Azure proje özellikleri yapılandırma, Azure projeniz için hello özellik iletişim kutuları aracılığıyla gerçekleştirilir. Eclipse'nın Proje Gezgini bölmesinde Hello projesine sağ tıklayın, **özellikleri**ve ardından hello **Azure** alt menüsünde.

![][ic719480]

Bu özellik sayfasında, aşağıdaki öğelerindeki hello değiştirebilirsiniz: 

* **Hizmet adı:** Azure projeniz için hello ad.
* **Derleme için:** toohello işlem öykünücüsü veya dağıtım tooAzure oluşturmak için başlangıç projesi olup ayarlanır.
* **Hedef işletim sistemi:** projenizi dağıtılan tooAzure olduğunda işletim sistemi toouse hello.
