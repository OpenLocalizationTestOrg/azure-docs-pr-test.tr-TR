# <a name="supported-postgresql-database-versions"></a>Desteklenen PostgreSQL veritabanı sürümleri
Azure veritabanı PostgreSQL için şu anda aşağıdaki sürümlerini destekler:

## <a name="postgresql-version-962"></a>PostgreSQL sürüm 9.6.2
Başvurmak [PostgreSQL belgelerine](https://www.postgresql.org/docs/9.6/static/release-9-6-2.html) geliştirmeleri ve PostgreSQL 9.6.2 düzeltmeler hakkında daha fazla bilgi edinmek için.

## <a name="postgresql-version-957"></a>PostgreSQL sürüm 9.5.7
Başvurmak [PostgreSQL belgelerine](https://www.postgresql.org/docs/9.5/static/release-9-5-7.html) geliştirmeleri ve düzeltmeler hakkında bilgi edinmek için PostgreSQL 9.5.7 içinde.

## <a name="managing-updates-and-upgrades"></a>Yönetme güncelleştirmeleri ve yükseltmeleri
Azure veritabanı PostgreSQL için ikincil sürüm güncelleştirmeleri düzeltme eki uygulama otomatik olarak yönetir. Şu anda, genel önizlemede ana sürüm yükseltme desteklenmiyor. Örneğin, PostgreSQL 9.6 PostgreSQL 9.5 ' yükseltme desteklenmiyor.
