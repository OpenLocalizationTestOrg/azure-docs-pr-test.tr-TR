# <a name="supported-browsers-and-devices-for-the-azure-portal"></a>Desteklenen tarayıcılar ve cihazlar için Azure portalı
Çalıştırabilirsiniz [Azure portal](https://portal.azure.com) tüm modern masaüstünde, aygıtları ve tarayıcılar tablet.

### <a name="supported-devices"></a>Desteklenen cihazlar
Azure portalı, modern PC'leri, Mac'leri ve tabletler harika çalışır. Bir mobil aygıttan Azure kaynaklarınızı yönetmek gerekiyorsa, iPhone ve Android'de Önizleme için kullanılabilir olan Azure uygulaması deneyin. Daha fazla bilgi için blog gönderisine bakın [Azure uygulaması genel Önizleme Tanıtımı](https://azure.microsoft.com/blog/azure-app-preview/).

### <a name="supported-browsers"></a>Desteklenen tarayıcılar
İşletim sistemi ile uyumlu en güncel tarayıcı kullanmanızı öneririz. Aşağıdaki tarayıcılarda desteklenir:

* Microsoft Edge (en son sürüm)
* Internet Explorer 11
* Safari (en son sürüm, yalnızca Mac)
* Chrome (en son sürüm)
* Firefox (en son sürüm)
