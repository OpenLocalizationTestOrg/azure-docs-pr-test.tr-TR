# <a name="connect-your-microsoft-azure-iot-raspberry-pi-3-starter-kit-to-the-remote-monitoring-solution"></a>Microsoft Azure IOT Raspberry Pi 3 Starter Kit Uzaktan izleme çözümüne bağlama

Bu bölümdeki öğreticileri Raspberry Pi 3 aygıtı Uzaktan izleme çözümüne bağlama öğrenin yardımcı olur. Tercih edilen programlama diline uygun öğretici seçin ve algılayıcı donanım, Raspberry Pi'yi ile kullanmak için kullanılabilir olup olmadığı.

## <a name="the-tutorials"></a>Öğreticiler

| Öğretici | Notlar | Diller |
| -------- | ----- | --------- |
| Sanal telemetriyi (Temel)| Algılayıcı verilerini benzetimini yapar. Tek başına Raspberry Pi'yi kullanır. | [C][lnk-c-simulator], [Node.js][lnk-node-simulator] |
| Gerçek algılayıcısı (Orta) | Raspberry Pi'yi bağlı BME280 algılayıcı verilerini kullanır. | [C][lnk-c-basic], [Node.js][lnk-node-basic] |
| Uygulama yazılımı güncelleştirmesi (Gelişmiş)| Raspberry Pi'yi bağlı BME280 algılayıcı verilerini kullanır. Raspberry Pi'yi uzak Bellenim güncelleştirmeleri sağlar. | [C][lnk-c-advanced], [Node.js][lnk-node-advanced] |
