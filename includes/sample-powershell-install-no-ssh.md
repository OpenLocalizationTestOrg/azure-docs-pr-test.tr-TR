Bu örnek, Azure PowerShell modülü sürüm 5.1.1 gerektiriyor veya sonraki bir sürümü. Sürümü bulmak için ` Get-Module -ListAvailable AzureRM` komutunu çalıştırın. Yüklemeniz veya yükseltmeniz gerekirse, bkz. [Azure PowerShell Modülü yükleme](/powershell/azure/install-azurerm-ps). 

Çalıştırma `Login-AzureRmAccount` Azure ile bir bağlantı oluşturmak için.
