
#### <a name="tooconfigure-remote-management-on-cloud-appliance"></a>Bulut uygulaması tooconfigure uzaktan yönetimi

1. StorSimple Cihaz Yöneticisi hizmetinizde **Cihazlar**’a tıklayın. Seçin ve aygıtları bağlı toohello hizmetinin hello listesinden, bulut uygulaması tıklayın.
    ![StorSimple bulut gereci seçme](./media/storsimple-8000-configure-remote-management-http-device/sca-remote-manage1.png)

2. Çok Git**ayarlar > Güvenlik** tooopen hello **güvenlik ayarları** dikey.

     ![StorSimple güvenlik ayarları](./media/storsimple-8000-configure-remote-management-http-device/sca-remote-manage2.png)

3. Toohello Git **uzaktan yönetimi** bölümü. **Uzaktan yönetim**  kutusuna tıklayın.
     ![StorSimple uzaktan yönetim](./media/storsimple-8000-configure-remote-management-http-device/sca-remote-manage3.png)

4. Merhaba, **uzaktan yönetimi** dikey penceresinde:

    1. **Uzaktan yönetimi etkinleştir** seçeneğinin etkin olduğundan emin olun.
    2. Merhaba tooconnect HTTPS üzerinden varsayılandır. HTTP kullanarak tooconnect seçebilirsiniz. HTTP üzerinden bağlanma yalnızca güvenilen ağlarda kabul edilebilir. HTTP’nin etkin olduğundan emin olun.
    3. Merhaba komutu dikey penceresinde hello üstündeki çubuğundaki **... Daha fazla** ve ardından **indirme sertifika** toodownload uzak yönetim sertifikası. Bu dosyayı hangi toosave bir konum belirtebilirsiniz. Tooconnect toohello bulut uygulaması kullanın, bu sertifika hello istemci veya konak makineye yüklenmesi gerekir.

        ![Uzaktan yönetim dikey penceresi](./media/storsimple-8000-configure-remote-management-http-device/sca-remote-manage4.png)
5. Tıklatın **kaydetmek** ve istendiğinde hello değişiklikleri onaylayın.
